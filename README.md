# 🐍 NEON SNAKE: ARCADE EDITION
A high-fidelity, cyberpunk-themed Snake game built with vanilla JavaScript. > Designed for a premium arcade experience with neon aesthetics, dynamic sound effects, and a built-in skin shop.

# ✨ KEY FEATURES
Cyberpunk Visuals: Smooth neon glow effects and high-contrast UI designed for large screens.

Dynamic Audio: Real-time synthesized sound effects for eating, moving, and game over states.

Skin Shop: Collect coins during gameplay to unlock unique snake skins:

Classic: The original emerald green.

Neon: High-voltage cyan with white highlights.

Lava: Molten orange with a burning glow.

Persistent Progress: High scores and unlocked skins are saved locally in your browser.

Audio Control: Built-in mute/unmute toggle for shared environments.

# 🚀 QUICK START
Download the index.html file from this repository.

Open it in any modern web browser (Chrome, Edge, Firefox).

Click the START GAME button and enter the neon grid.

# 🛠️ TECHNICAL SETUP
This project is built using:

HTML5 Canvas: For high-performance 2D rendering.

Web Audio API: To generate retro-futuristic synth sounds without external assets.

Local Storage: For saving game data across sessions.
